## Getting Started

### Local

First, run the development server:

```bash
yarn dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

### On Vercel Cloud

See live demo on [vercel cloud](https://f43fcc6d.vercel.app/)
